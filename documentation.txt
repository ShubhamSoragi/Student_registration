## Documentation

This project is a simple HTML/JS/CSS starter template. It provides a basic structure for creating a web page with HTML, styling it with CSS, and adding interactivity with JavaScript.

### HTML

The HTML file (`index.html`) contains the basic structure of the web page, including the `<head>` and `<body>` elements. The `<head>` element contains the title of the page and a link to the CSS file. The `<body>` element contains the content of the page, which is currently just a placeholder text.

### CSS

The CSS file (`style.css`) defines the styles for the web page. It includes styles for the body, headings, and paragraphs.

### JavaScript

The JavaScript file (`main.js`) contains the code that adds interactivity to the web page. It includes a function that changes the text of the placeholder text when the user clicks on it.
